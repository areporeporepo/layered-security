# The Playbook — Do This, Not Read This

No theory on this page. Every line is an action you can check off. Start at the top.

---

## NOW — 60-second safety triage

- [ ] **Is anything physical or threatening happening, or is someone hurting you where you live?**
  - **Yes →** call **911** if immediate; otherwise **National DV Hotline 1-800-799-7233** (text START to 88788). Do this before anything else on this page.
  - **No →** continue.
- [ ] **Are you in crisis / can't cope right now?** Call or text **988**, or text **HOME to 741741**.
- [ ] If neither, breathe once, slowly. Then keep going.

---

## TODAY — the first 24 hours

- [ ] Open `incidents.csv` (or any notes app). Log **one** thing that happened: date, time, what happened (facts only), severity 1–5.
- [ ] Write the facts like a camera would — no "they did it because…". Put guesses in the *inference* column.
- [ ] Save one piece of evidence for it: a screenshot (with timestamp/URL), a photo, a name. If none, write "none."
- [ ] Identify your **highest-stakes layer** right now (safety > livelihood > everything else).
- [ ] Tell **one** real person something true about what's happening. Out of your head, into the world.

---

## THIS WEEK

- [ ] Log every incident as it happens — even one line each. Don't batch from memory.
- [ ] Run `python3 score.py` once. Look at the AEI number and which layer is highest.
- [ ] Pick the **one** layer with the highest score/stakes. Ignore the rest this week.
- [ ] Do that layer's playbook below.
- [ ] Set up one piece of passive evidence (door camera, call recording where legal, email-to-self after incidents).

---

## PER-LAYER PLAYBOOKS — do the one that's worst first

### 🏠 Home / Physical (highest priority)
- [ ] File a police report for each incident — it's a timestamp, not a fix. Keep the report numbers.
- [ ] Re-key / change locks. Add an entry sensor + a camera on the door.
- [ ] Photograph anything tampered with **before** you touch it.
- [ ] Ask a lawyer about a **protective/restraining order** — the one tool with real teeth. Bring your log.

### 💼 Workplace
- [ ] Put every incident in writing (email yourself, dated).
- [ ] File internally with HR/ombuds in writing, citing the company's own policy.
- [ ] Check the deadline and file with the **EEOC (eeoc.gov)** or your state agency — don't sit on it.
- [ ] Book a **free consult** with an employment lawyer (many work contingency). Bring the log.

### 💬 Social / Online
- [ ] Screenshot everything **with timestamps and URLs** before blocking.
- [ ] Report threats/impersonation/doxxing to the platform's safety team.
- [ ] If there are threats, file a police report to create a record.

### 🏘️ Neighborhood / Public
- [ ] Log dates/times of recurring incidents (pattern > any single event).
- [ ] Use the non-emergency police line to create a documented record.
- [ ] Put any landlord/HOA on notice in writing, dated.
- [ ] For stalking specifically: use **SPARC** (search "SPARC stalking") to build a safety plan.

### 🏛️ Institutional (a body that won't act)
- [ ] Document the non-response with the same discipline as the original incident.
- [ ] Escalate to the body **above** it (state agency, licensing board, inspector general).
- [ ] Bring the whole package to a lawyer — the non-response is now evidence too.

---

## THE LOOP — repeat until resolved

- [ ] **Log** every incident (facts + severity + evidence).
- [ ] **Score** weekly (`python3 score.py`). Watch the trend.
- [ ] **Escalate** the highest layer to a channel with real power (lawyer, protective order, agency, footage).
- [ ] **Protect yourself**: breath, movement, sleep, one human. Every day.
- [ ] When a layer is handled, move to the next highest. One at a time.

---

> **The only rule:** a feeling can't be acted on; a dated total can. Convert one into the other,
> one logged line at a time — and don't carry the total alone.

Need the *why* behind any of this? It's in the [README](README.md) and `docs/`. But you don't
need it to start. Start at NOW.
